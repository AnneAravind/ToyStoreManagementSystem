# ToyStoreManagementSystem
I developed a Toy Store Management application using Java Servlets that supports CRUD operations for toy records, including adding, updating, viewing, and deleting. The # Toy Store Management System

A web-based Toy Store Management System built using Java Servlets and JDBC. The application allows users to manage toy records through a simple interface and demonstrates core Java, server-side programming, database connectivity, and CRUD operations.

## Features
- Add new toy records
- View all available toys
- Update toy details
- Delete toy records
- Manage toy information such as name, price, category, and quantity
- Store and retrieve toy data using a database
- User-friendly web interface

## Technologies Used
- Java
- Java Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat

## Concepts Used
- CRUD Operations
- Server-Side Programming
- JDBC Database Connectivity
- MVC Architecture
- Form Handling
- Exception Handling

## How to Run
1. Clone this repository.
2. Create the required MySQL database and tables.
3. Update JDBC database connection details in the project.
4. Deploy the application on Apache Tomcat.
5. Open the application in a web browser.
